This is the API documentation of StudyGuide.

Modules

StudyGuide is logically separated into five modules:
  • constraint - contains the constraint class hierarchy, along with events fired and caught by the constraints.
  • controller - controllers translate user actions from the view into operations on the model.
  • model - basic data structures that keeps all the user's data.
  • persistence - reads and writes the data in the model to persistent storage (e.g. hard disk).
  • view - contains display elements for the graphical user interface and various GUI utility classes. Most of the view code is hidden in *.fxml JavaFX 8 files in src/main/resources.
  • weld - utility classes for Context and Dependency Injection (CDI) using Weld.
